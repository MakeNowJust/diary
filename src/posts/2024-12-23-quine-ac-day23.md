# DAY 23

[Erg](https://erg-lang.org)というプログラミング言語があります。
Pythonのバイトコードに変換される言語みたいです。
あとは型はまあまあちゃんと付いています。
文法はあまりしっくりこなかったです。

Quineはフォーマットで適当にやりました。
正直ドキュメントが無いので何ができて何ができないのかよく分からなかった。

```hs
s = """s = ""{0:c}{1}{0:c}""
print! s.format(34, s)"""
print! s.format(34, s)
```

実行は `erg` コマンドで。

```console
$ erg quine.er | diff quine.er -
```
